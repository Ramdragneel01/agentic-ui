# Contributing to agentic-ui

## Prerequisites

- Node.js 20+
- npm 9+

## Local Workflow

```bash
npm install
npm run typecheck
npm run build
npm run storybook
```

## Pull Request Checklist

1. Component API changes documented in stories and README.
2. Accessibility behavior manually verified.
3. docs/FUTURE-CLARIFICATIONS.md updated for non-obvious decisions.
4. CHANGELOG.md updated for user-visible behavior changes.
